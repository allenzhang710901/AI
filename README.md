# 从 0 开始的 AI：一个可运行的中文命令行助手

这个项目提供了一个**不依赖云 API** 的最小 AI 示例：

- 使用纯 Python 实现一个朴素贝叶斯文本分类器。
- 用少量中文语料训练「意图识别」模型。
- 在命令行里进行对话，根据识别出的意图给出回复。

> 目标：帮助你理解“从 0 开始写一个 AI”的最小闭环：
> 数据 → 训练 → 预测 → 交互。

---

## 1. 环境要求

- Python 3.9+

无需额外第三方依赖。

---

## 2. 快速开始

```bash
python main.py
```

启动后输入中文句子，例如：

- `你好`
- `帮我算一下 18 * 7`
- `今天北京天气怎么样`
- `给我推荐一本书`

输入 `退出` 结束。

---

## 4. 你可以继续扩展的方向

1. 增加训练数据（每个意图更多样本）。
2. 加入分词与停用词处理。
3. 将规则回复改成函数调用（查天气 API、写待办等）。
4. 增加 Web 界面（Flask/FastAPI + 前端）。
5. 引入向量检索（RAG）支持你的私有知识库。

---

## 5. 为什么这个例子叫“从 0 开始”

因为它展示了 AI 最核心的工程思维：

- 先定义任务（意图分类）
- 准备数据（标注语料）
- 选择模型（朴素贝叶斯）
- 训练与评估（看预测置信度）
- 接入交互（CLI）

这条路径你掌握后，再换成更强的模型（深度学习、LLM API）会非常自然。
